JavaScript

JavaScript é a linguagem responsável por dar comportamento e interatividade às páginas web, permitindo responder a ações do usuário (como cliques e digitação), manipular o conteúo HTML em tempo real e se comunicar com servidores;

Em resumo, enquanto HTML estrutura e CSS estiliza, o JavaScript faz a página "funcionar".
